# 📖 Project Overview

The **Airline Reservation & Management System** is a desktop application developed to simplify airline operations by digitalizing passenger management, flight scheduling, booking processes, and reservation management.

The system enables administrators to efficiently manage flights, customer records, ticket bookings, and cancellations while providing an organized interface for handling airline operations.

This project demonstrates Object-Oriented Programming concepts, Java GUI development, database connectivity, and CRUD operations.

---

# 🎯 Purpose

The purpose of this system is to replace manual airline reservation processes with a computerized management system that improves efficiency, reduces booking errors, and enhances customer service.

The application allows administrators to:

- Manage flights
- Register passengers
- Book airline tickets
- Cancel reservations
- Search passenger records
- Maintain airline data efficiently

## Clone Repository

```bash

```

Move into project


Open the project in:

Configure MySQL database and update the JDBC connection settings before running the application.
